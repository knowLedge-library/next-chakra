## 如何选择 SSR 还是 SSG？

- 如果页面内容真动态(例如，来源数据库，且经常变化)， 使用 getServerSideProps 方法的 SSR。
- 如果是静态页面或者伪动态(例如，来源数据库，但是不变化)，可以酌情使用 SSG。
